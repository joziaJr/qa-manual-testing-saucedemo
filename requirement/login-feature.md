# Login Feature Requirement Analysis

## System Under Test
SauceDemo Website

URL:
https://www.saucedemo.com

## Feature
User Login

## Login Page Components
- Username field
- Password field
- Login button
- Error message area

## Functional Requirement

FR-01  
User must be able to input username.

FR-02  
User must be able to input password.

FR-03  
System should allow login with valid credentials.

FR-04  
System should reject login with invalid credentials.

FR-05  
User who successfully logs in should be redirected to the product inventory page.

FR-06  
If login fails, system should display an error message.

## Assumptions

User must have a valid account to login.

Example valid accounts provided by the system:
- standard_user
- locked_out_user
- problem_user
- performance_glitch_user
